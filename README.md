# Figma Clone (Placement UI)

This project is a pixel-perfect, responsive web UI inspired by Figma's design system. It demonstrates how to:

- Separate HTML, CSS, and JavaScript for clean code organization
- Build a modern, responsive layout for both desktop and mobile
- Implement a hero slider with navigation and dot indicators
- Use semantic HTML and accessible controls

## Project Structure

- `index.html` — Main HTML file (links to external CSS and JS)
- `style.css` — All styles for layout, responsiveness, and theming
- `script.js` — JavaScript for slider functionality and interactivity

## How to Use

1. Open `index.html` in your browser.
2. Explore the desktop and mobile layouts by resizing the window.
3. Use the slider arrows or dots to navigate between hero slides.

## Features

- Clean separation of structure (HTML), style (CSS), and behavior (JS)
- Fully responsive: adapts to desktop and mobile
- Accessible navigation controls
- Easy to extend for your own placement or design showcase
